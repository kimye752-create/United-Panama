```
db_connector.ts (베이스 — 독립)
    ↑
BaseCrawler.ts (db_connector 사용)
    ↑
ApiCrawler.ts / StaticCrawler.ts (BaseCrawler 상속)
    ↑
pa_worldbank.ts, pa_pubmed.ts (ApiCrawler 상속)
pa_ita.ts, pa_kotra.ts, pa_motie.ts (StaticCrawler 상속)
    ↑
preload_macro.ts (5개 크롤러 순차 실행)
    ↑
pa_static_macro.yml (preload_macro.ts 실행)
```

### 핵심 타입 시그니처
```typescript
// db_connector.ts
export type MarketSegment = "public" | "private" | "macro";
export interface PanamaCommonPhase1 {
  id?: string;
  product_id: string;
  market_segment: MarketSegment;
  fob_estimated_usd: null;  // 타입 레벨 강제
  confidence: number;
  crawled_at: string;
}
export type InsertRowResult = 
  | { ok: true } 
  | { ok: false; message: string; cause?: unknown };
export async function insertRow(row: PanamaPhase1InsertRow): Promise<InsertRowResult>;

// BaseCrawler.ts
export type PaSourceType = "static_pre_loaded" | "llm_realtime";
export type CrawlRowData = { product_id: string } & Record<string, ...>;
export type CrawlRunResult = 
  | { ok: true; inserted: number }
  | { ok: false; message: string; cause?: unknown; inserted: number };
export abstract class BaseCrawler {
  constructor(name, sourceType, market_segment, baseConfidence);
  protected abstract crawl(): Promise<CrawlRowData[]>;
  async run(): Promise<CrawlRunResult>;
}
```

- **검증 메커니즘**: 추출값 vs 예상값 일치 검증, 불일치 시 step-back 트리거
- **dmax 재시도 상한**: 무한 루프 방지
- **반영 위치**: 기법 ② Wrapper Maintenance 서브 메커니즘

### 제로샷 + 소형 모델 논문
- **5단계 HTML 전처리 파이프라인**:
  1. HTML 파싱 (Cheerio)
  2. 노이즈 제거 (script, style, 주석)
  3. DOM 구조 단순화 (class/id/data-* 속성 제거)
  4. 텍스트 블록 추출 (텍스트 밀도 지표 기반)
  5. 구조화 토큰 변환 (<h1>→[TITLE], <p>→[PARAGRAPH])
- **텍스트 밀도 = 실제 텍스트 길이 / 전체 문자 길이**
- **효과**: 소형 모델(Claude Haiku) 토큰 비용 70~80% 절감 + 정확도 향상
- **반영 위치**: 신규 기법 ⑧ HTML Preprocessing Pipeline
