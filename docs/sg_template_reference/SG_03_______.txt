▸ 기업 개요

② 파이프라인  CPHI 등록 제품에 Contrast Media Dyes, Highly Potent APIs 포함. Gadobutrol 계열 조영제 원료 취급 이력 보유. 유사 macrocyclic 화합물 제조 역량 확인.

③ 제조소 보유  인도 내 GMP 인증 제조 시설 보유. EU GMP 및 US FDA 승인 제조소 운영. 위탁 제조(CMO) 및 자체 제조 병행 가능.

④ 수입 경험  싱가포르 포함 APAC 지역 의약품 수입·유통 경험 보유. 다수의 글로벌 제약사에 API 공급 실적 있으며, Singapore 거래 레퍼런스 확인 가능.

2. CHENGDU BRILLIANT PHARMACEUTICAL GROUP  |  중국 · Ingredients

▸ 기업 개요

쓰촨 르난 제약의 모회사로, 의약 중간체 및 원료의약품(API) 연구개발·제조·판매를 전문으로 하는 고기술 기업. 2013년 설립 이후 조영제·항암제·CNS 계열 API 분야에서 빠르게 성장하고 있으며, CPHI 등록 제품에 Gadobutrol을 직접 포함하고 있어 가도부트롤 원료 공급자로서의 적합성이 가장 높은 후보군입니다.

▸ 추천 이유

① 매출 규모  설립 초기 기업으로 정확한 매출 공개 데이터가 제한적이나, 그룹사 기준 연간 매출 추산 USD 50~80M 수준. 중국 내 고기술 인증(国家高新技术企业) 보유.

② 파이프라인  CPHI 공식 등록 제품에 Gadobutrol 직접 포함. 추가로 Iopromide, Iodixanol, Ioversol 등 조영제 계열 원료 취급. Gadvoa Inj.와 동일 성분 API 공급 역량 보유로 파이프라인 적합성 최상위.

③ 제조소 보유  청두 하이테크 산업개발구 내 자체 제조 시설 보유. 중국 NMPA 인증 취득. API 제조 및 중간체 합성 전문 설비 운영.

④ 수입 경험  싱가포르 직접 수입 경험이 공개 정보상 미확인. 다만 중국 내 다수의 글로벌 제약사에 API 수출 실적 있으며, 싱가포르 바이어와의 연결을 통한 간접 진출 가능성 존재.

⑤ 약국 체인 운영  약국 체인 운영 없음. 원료의약품(API) 전문 공급사로, 완제의약품 유통보다는 한국유나이티드제약의 Gadvoa 제조에 필요한 원료 안정 공급 파트너로서의 가치가 더 높음.
